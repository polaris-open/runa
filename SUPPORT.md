# Support

Runa is **Draft v0.1 — early concept / runnable skeleton**. There is no support
SLA, and many features are intentionally not built yet.

## Where to start

1. Read [START_HERE.md](START_HERE.md) and pick the profile that matches you.
2. Check [README.md](README.md) for what Runa does and does not do today.
3. Browse the [docs/](docs/) directory, especially
   [docs/mvp.md](docs/mvp.md) for the current scope.

## Getting help

- **Questions / ideas:** open a GitHub Discussion or issue on
  [polaris-open/runa](https://github.com/polaris-open/runa).
- **Bugs:** open an issue with steps to reproduce and your Python version.
- **Security concerns:** follow [SECURITY.md](SECURITY.md) — do not open a public
  issue.

Organization-wide support defaults live in
[`polaris-open/.github`](https://github.com/polaris-open/.github).

## Please remember

- Do not include real data, secrets, or PII in issues or examples.
- "It does not answer questions about my notes" is expected: `ask` is not
  implemented in v0.1.
